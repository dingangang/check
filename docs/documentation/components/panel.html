Panel

一个组合的面板，为复杂的控制器而设置

{% include meta.html colors=false sizes=false variables=true %}

panel是一下几种类型的容器:

  • panel-heading作为第一个子元素
  • panel-tabs作为导航
  • panel-block可以包含其他的元素，比如：
    • control
    • input
    • button
    • panel-icon

panel-block可以作为一个链接标签<a>或作为一个带有多选框的标签<label>。
