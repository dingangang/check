Tabs

简单的水平响应式导航tabs，拥有不同的样式

{% include meta.html since="0.4.4" colors=false sizes=true variables=true %}

Tabs仅仅需要一个tabs容器和一个<ul>列表。
默认的tabs样式有一个单独的底部边线。

{% include snippet.html content=tabs_example horizontal=true more=true %} {% include anchor.html name="对齐（Alignment）" %}

要定位列表，只需要在在.tabs容器上使用is-centered、is-right修饰器 ：

{% include snippet.html content=tabs_centered_example horizontal=true more=true %} {% include snippet.html content=tabs_right_example horizontal=true more=true %} {% include anchor.html name="图标" %}

你可以使用任意的Font Awesome 图标。

{% include snippet.html content=tabs_icons_example horizontal=true more=true %} {% include anchor.html name="尺寸" %}

你可以在3种不同尺寸中进行选择: is-small is-medium and is-large。

{% include snippet.html content=tabs_small_example horizontal=true more=true %} {% include snippet.html content=tabs_medium_example horizontal=true more=true %} {% include snippet.html content=tabs_large_example horizontal=true more=true %} {% include anchor.html name="样式" %}
如果你想要使用更简单的带边框的样式，只需要添加is-boxed修饰器。
{% include snippet.html content=tabs_boxed_example horizontal=true more=true %}

如果你想要互不相交的tabs，使用is-toggle修饰器。

{% include snippet.html content=tabs_toggle_example horizontal=true more=true %}

如果你想要tabs占据可用的全宽，那么就使用is-fullwidth修饰器。

{% include snippet.html content=tabs_fullwidth_example horizontal=true more=true %} {% include anchor.html name="拼接" %}

你可以拼接不同的修饰器。比如说你可以使用居中的盒状 tabs,或者全宽互相分隔的tabs。

{% include snippet.html content=tabs_centered_boxed_example horizontal=true more=true %} {% include snippet.html content=tabs_toggle_fullwidth_example horizontal=true more=true %} {% include snippet.html content=tabs_centered_boxed_medium_example horizontal=true more=true %} {% include snippet.html content=tabs_toggle_fullwidth_large_example horizontal=true more=true %} {% include variables.html type='组件' %}
