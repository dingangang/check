卡片（Card）

一个全方位的灵活组合的组件

{% include meta.html variables=true colors=false sizes=false %}

card组件由好几种不同的元素构成，你可以混合或者匹配使用：

  • card：主容器
    • card-header：带阴影的水平栏
      • card-header-title：居左的加粗文本
      • card-header-icon：图标占位符
    • card-image：全宽的响应式图片容器
    • card-content：一种可以使用任意 元素的多用途容器
    • card-footer：一个水平的多用途控制器
      • card-footer-item：一个可重复使用的项目列表
New 0.5.3

你可以让通过添加is-centered修饰器来让card-header-title居中。
