分页（Pagination）

一个响应式的、好用、灵活的分页

{% include meta.html colors=false sizes=true variables=true %}

分页组件包含了几种元素：

  • pagination-previous和pagination-next来增量导航
  • pagination-list用来展示页面项目：
    • pagination-link用于展示页面数字
    • pagination-ellipsis用于区域分割

所有的元素都是可选的，你可以按自己的医院构建分页。
