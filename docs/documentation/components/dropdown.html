下拉框（Dropdown）

一个对于可发现内容的交互下拉菜单

{% include meta.html new=true since="0.4.4" colors=false sizes=false variables=true %}

dropdown组件是下拉按钮和下拉菜单的容器。

  • dropdown是主容器
    • dropdown-trigger是一个button的容器
    • dropdown-menu可以切换的菜单，默认是隐藏的
      • dropdown-content下拉菜单盒子，带有白色背景和阴影
        • dropdown-item下拉菜单单个项目的容器，可以包括一个a或者一个div
        • dropdown-divider一个水平分隔线来分离下拉菜单

{% include anchor.html name="（悬停切换）Hoverable or Toggable" %}

dropdown组件有2个可选类

  • is-hoverable：当鼠标悬停（hovering）在dropdown-trigger时，下拉菜单就会显示
  • is-active：下拉才当会一直显示

当:hover可以使用，is-active类可以供用户通过JavaScript来控制下拉菜单的显示与否。.
