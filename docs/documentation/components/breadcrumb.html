面包屑

一个简单的面包屑组件来提升你的浏览体验

{% include meta.html since="0.4.3" variables=true colors=false sizes=true %}

面包屑组件只需要一个.breadcrumb容器和一个ul列表

分割线是自动生成的，文本在li标签的::before伪类中生成。

你可以通过is-active修饰器来标记当前页面的li标签。它将禁用内部链接的导航。

{% include snippet.html content=breadcrumb_example horizontal=true clipped=true %} {% include anchor.html name="对齐" %}

这里为面包屑.breadcrumb容器提供了对齐修饰器，使用is-centered修饰器来居中显示，也可以使用is-right修饰器来居右。

{% include snippet.html content=breadcrumb_centered_example horizontal=true clipped=true %} {% include snippet.html content=breadcrumb_right_example horizontal=true clipped=true %} {% include anchor.html name="图标" %}

你可以使用任意的Font Awesome 图标.

{% include snippet.html content=breadcrumb_icons_example horizontal=true clipped=true %} {% include anchor.html name="可选分隔线" %}

你可以在4种可选的分隔线: has-arrow-separator has-bullet-separator has-dot-separator和has-succeeds-separator中任意选取使用。

{% include snippet.html content=breadcrumb_arrow_example horizontal=true clipped=true %} {% include snippet.html content=breadcrumb_bullet_example horizontal=true clipped=true %} {% include snippet.html content=breadcrumb_dot_example horizontal=true clipped=true %} {% include snippet.html content=breadcrumb_succeeds_example horizontal=true clipped=true %} {% include anchor.html name="尺寸" %}

你可以在3种可选尺寸: is-small is-medium和is-large中任意选取使用。
