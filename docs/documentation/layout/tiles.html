{% endhighlight %} {% include anchor.html name="修饰器（Modifiers）" %}

tile元素拥有16个修饰器:

  • 3个前后文修饰器
    • is-ancestor
    • is-parent
    • is-child
  • 1个方向修饰器
    • is-vertical
  • 12水平尺寸修饰器
    • 从is-1
    • 到is-12

只要你想要添加内容到瓷砖片内, 只需要:

  • 添加任意 你想要的类,比如box
  • 添加is-child修饰器到瓷砖片

{% endhighlight %} {% include anchor.html name="嵌套需求（Nesting requirements）" %}
至少需要3层的嵌套

你需要至少3个层级：

{% highlight markdown %} tile is-ancestor | └───tile is-parent | └───tile is-child {% endhighlight %}
但是你可以使用更多层的嵌套

你可以使用更多的层级，并且可以混合使用

{% highlight markdown %} tile is-ancestor | ├───tile is-vertical is-8 | | | ├───tile | | | | | ├───tile is-parent is-vertical | | | ├───tile is-child | | | └───tile is-child | | | | | └───tile is-parent | | └───tile is-child | | | └───tile is-parent | └───tile is-child | └───tile is-parent └───tile is-child {% endhighlight %}
