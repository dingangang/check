容器（Container）

一个简单的容器来让你的内容居中

.container类可以在任意内容中使用，但是更多的情况会是成为以下元素的的直接子元素:

  • .navbar
  • .hero
  • .section
  • .footer

容器的断点会有一个偏移量由变量$container-offset来定义。默认的值为60px.

容器的展示情况如下所示:

  • 当$desktop + $container-offset = >= 1068px它会有一个最大宽度（maximum width） 960px并且会是水平居中.
  • on $widescreen + $container-offset = >= 1260px它会有一个最大宽度（maximum width） 1152px.
  • on $fullhd + $container-offset = >= 1452px它会有一个最大宽度（maximum width） 1344px.

变量$gap 的默认值为24px，但是它可以被修改。

选择960, 1152和1344这几个值的原因是他们可以被12和16除尽。
