水平层（Level）

一个多用途的水平放置层，其中可以放置几乎其他的所有元素

其结构如下图所示:

  • level:主容器
    • level-left左侧子元素，这个元素是必须的，哪怕是空的也需要放置。
    • level-right右侧子元素。
      • level-item独立元素

在level-item元素中,你可以放置任意 想放置的元素包括: 标题、标签或者是单纯的文字。无论你放置什么元素到Bulma水平层（level)中，它们总是垂直居中。
