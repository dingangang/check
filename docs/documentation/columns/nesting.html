嵌套（Nesting columns）

一种简单的方式来构建响应式列布局

你可以嵌套列来获取更多的灵活性。你只需要按照下面的结构布局:

  • columns: 顶层列布局容器
    • column
      • columns: 嵌套列容器布局
        • column 更多...

嵌套布局和多行列布局（multiline columns） 是在HTML的顺序：所有的blue列会一直在red之前 。调整显示器到一个更小的视口来观察变化。
