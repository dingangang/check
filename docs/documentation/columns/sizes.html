列尺寸（Column sizes）

单独的定义每一列的尺寸

如果你需要单独改变某一列的尺寸，你可以使用以下类:

  • is-three-quarters
  • is-two-thirds
  • is-half
  • is-one-third
  • is-one-quarter

其他的会自动填充 剩余的空白部分。

  • is-narrow-tablet
  • is-narrow-desktop
