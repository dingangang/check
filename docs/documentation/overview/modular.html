模块化（Modular）

只需要引入你所需的

Bulma包含39个.sass文件，每一个都可以独立引入。

比如说你只想使用Bulma的columns.样式
这个文件位于bulma/sass/grid文件夹.
主要简单的importutilities依赖,然后直接引入你需要的文件:

{% highlight sass %} @import "bulma/sass/utilities/_all" @import "bulma/sass/grid/columns" {% endhighlight %}

现在你就可以直接使用类.columns (for the container)和.column:

{% highlight html %}
1
2
3
4
5
{% endhighlight %}

如果你只希望使用button样式呢?

{% highlight sass %} @import "bulma/sass/utilities/_all" @import "bulma/sass/elements/button.sass" {% endhighlight %}

你现在可以使用所有的.button类以及其衍生修饰器:

  • .is-active
  • .is-primary, .is-info, .is-success...
  • .is-small, .is-medium, .is-large
  • .is-outlined, .is-inverted, .is-link
  • .is-loading, [disabled]
