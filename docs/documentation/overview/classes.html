类（Classes）

Bulma是一个简单的CSS类的集合。写你所想的HTML。

Bulma是一个CSS框架，所以最后的输出结果是一个单独的 .css文件：
https://github.com/jgthms/bulma/blob/master/css/bulma.css

因为Bulma仅仅包括了CSS类，所以你所写的HTML文件不会影响页面风格。这就是.input单独作为一个类（class）存在的原因，所以你可以选择<input type="text">元素来进行样式设定.

Bulma通常为标签直接设定样式两次:

  • generic.sass定义页面基本风格
  • .content class 可以在所有的文本内容,例如 WYSIWYG
