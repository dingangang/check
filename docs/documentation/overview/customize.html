使用Sass来进行定制

使用一些简单的 变量集合 来定制 自己的主题

如果你对 Sass 比较熟悉，并且想通过自己的颜色和变量来 定制 Bulma , 只需要通过 npm 来安装Bulma :

1

下载 项目源文件:

{% highlight bash %}npm install bulma{% endhighlight %}
或者从GitHub库下载: https://github.com/jgthms/bulma

2

设定自己的变量

添加自己的颜色、字体等来覆盖Bulma的默认样式。首先在项目文档中新建一个app.sass与bulma.sass同目录，app.sass代码如下：

{% highlight sass %}{{ sass }}{% endhighlight %}
在项目文件中package.json，修改"build-sass": "node-sass --output-style expanded --source-map true app.sass css/bulma.css"。即指定对应的编译文件为用户自定义的app.sass。然后运行
{% highlight bash %}npm run build{% endhighlight %}

即可获取定以后的bulma.css

3

对比结果: 定制前和定制后

因为$blue 已经被更新,所以$blue-invert 会通过函数$blue-invert: findColorInvert($blue)自动计算生成,$blue-invert现在是黑色的而不是白色。
