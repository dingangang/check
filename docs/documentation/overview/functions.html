函数（Functions）

工具函数，用来动态计算颜色和其他值

Bulma使用了3个定制函数来辅助定义值和颜色:

  • powerNumber($number, $exp):计算暴露于另外一个数字的值。返回一个数字。
  • colorLuminance($color):定义一个颜色属于暗还是明，返回一个0~1之间的小数，当数值<=0.5时是暗，当数值>=0.5时是明
  • findColorInvert($color):返回70%透明的黑色或者100%不透明的白色，取决于颜色的亮度。
{% include anchor.html name="The findColorInvert() function" %}

函数findColorInvert($color)输入值是一种颜色输出是黑色 rgba(#000, 0.7)或者white #fff:

  • 若colorLuminance($color) > 0.55,则返回rgba(#000, 0.7)
  • 否则返回为#fff

其目的是为了确保输入颜色作为背景色情况下的文本 易读性

#3273dc 0.23119 #fff Button

#ffb3b3 0.61796 rgba(0,0,0,0.7) Button
#ffbc6b 0.63053 rgba(0,0,0,0.7) Button
hsl(294, 71%, 79%) 0.5529 rgba(0,0,0,0.7) Button

当颜色亮度接近0.55阈值,可以重写findColorInvert()函数,也可以手动指定颜色。
比如说当颜色是 的亮度值是0.5529此时，可以通过函数判定也可以手动指定，两者皆可:

with findColorInvert() $purple-invert: findColorInvert($purple) rgba(0,0,0,0.7) Button
with manual setting $purple-invert: #fff #fff Button
