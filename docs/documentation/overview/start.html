开始使用

你只需要 一个 CSS 文件 就可以使用 Bulma

有几种方式来 开始使用 Bulma。有以下几种方式可以选择:

  1. 使用 npm 来安装Bulma项目
  2. 使用 CDN 来获取 Bulma 样式表
  3. 使用 GitHub 库 来获取最新的开发版本

1

使用 NPM (推荐 ):

{% highlight bash %}npm install bulma{% endhighlight %}

2

使用 cdnjs CDN
https://cdnjs.com/libraries/bulma

3

从 GitHub repository 下载
https://github.com/jgthms/bulma/tree/master/css
