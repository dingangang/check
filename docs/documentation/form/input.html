New! 0.5.3

如果你使用readonlyHTML属性,这个输入框的样式与其他的一致，但是不可编辑也没有阴影。

如果你同时 添加了is-static修饰器,则会去除背景色，边框和水平内边距，同时会保持垂直空间。所以你也可以很方便的对齐其他文本。比如水平表单。

在空间容器中你可以拓展2个修饰器:

  • has-icons-left
  • 一同/或者 has-icons-right

你同样需要为icon添加修饰器:

  • 如果使用has-icons-left则添加icon is-left
  • 如果使用has-icons-right则添加icon is-right
