表单控件（Form controls）

所有基本的表单控件在设计上都保持了一致性

下面的表单控件类都是支持的:

  • .label
  • .input
  • .textarea
  • .select
  • .checkbox
  • .radio
  • .button
  • .help

所有的表单控件都需要包含在.control容器内。
当把一些表单控件都包含到表单中时, 可以使用.field类来作为一个容器，以此来保证一致的间隔。

{% include snippet.html content=example %} {% include anchor.html name="表单域（Form field）" %}

field是作为以下元素的容器:

  • 文本类label
  • 表单类control
  • 可选的类help文本
{% include snippet.html content=field_example %}

这个容器来确保恒定的间隔。

{% include snippet.html content=fields_example %} {% include anchor.html name="表单控件（Form control）" %}

control是一个多功能的类，用于增加单个表单控件。因为它作为一个登高的控制元素,它只能包含下列元素:

  • input
  • select
  • button
  • icon
{% include snippet.html content=control_input_example %} {% include snippet.html content=control_select_example %} {% include snippet.html content=control_button_example %} {% include anchor.html name="使用图标（With icons）" %}

你可以添加下面中的修饰器在contorl容器上:

  • has-icons-left
  • 一同/或者has-icons-right

你也需要添加一个修饰器到icon元素:

  • 当使用has-icons-left时，添加icon is-left
  • 当使用has-icons-right时，添加icon is-right

input元素的尺寸会定义icon的容器尺寸。

{% include snippet.html content=icons_example clipped=true %}

你也可以将图标添加到下拉菜单。

{% include snippet.html content=select_icons_example %}

如果control中存在图标，Bulma会确保图标居中，无论输入框或 图标的尺寸。

按钮列表

如果你只是需要列表 组, 可以尝试新的buttons 类，你可以直接创建多行按钮列表。

{% include anchor.html name="（水平表单）Horizontal form" %}

如果你需要水平的表单控件，添加is-horizontal修饰器到field容器, 其中可以添加:

  • field-label 作为侧标签
  • field-body作为input/select/textarea的容器

你可以使用is-grouped或has-addons来修饰其子元素。

{% include snippet.html content=horizontal_form_example horizontal=true more=true %}

为了防止标签垂直对齐时的尺寸问题,.field-label类拥有4个尺寸修饰器:

  • .is-small
  • .is-normal可以对任意.input或.button使用
  • .is-medium
  • .is-large
