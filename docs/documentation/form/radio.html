单选框（Radio）

基于浏览器原生样式设计单选框按钮

{% include meta.html colors=false sizes=false variables=false %}

radio只对原生的<input type="radio">元素做了简单的包装，特意保持了其无修饰的样式，为了避免多种浏览器之间的差异而影响用户体验。

确保单选框的name属性有相同的值。
