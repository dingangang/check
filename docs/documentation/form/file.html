文件（File upload）

订制的文件上传输入框，无JavaScript

{% include meta.html experimental=true since="0.5.1" colors=true sizes=true variables=true %}

.file元素是一个简单的包括<input type="file">的互动标签，它包含几个子元素:

  • .file元素是主容器
    • .file-label是实际的交互性元素和点击事件的触发部分。
      • .file-input是原生的文件输入框，处于样式的关系进行了隐藏。
      • .file-cta是上传的代替性按钮。
        • .file-icon可选择的上传图标。
        • .file-label是文本——"Choose a file…"
      • .file-name是选中文件名称的容器。
