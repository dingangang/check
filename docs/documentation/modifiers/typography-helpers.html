{% include anchor.html name="文本位置（Alignment）" %}

当你需要使用align属性的时候，可以应用以下4个文本位置辅助类:

Class Alignment
has-text-centered 让文本居中（centered）
has-text-justified 让文本铺满（justified）
has-text-left 让文本左对齐
has-text-right 让文本右对齐

{% include anchor.html name="文本转化（Text transformation）" %}

你可以使用3种文本转化修饰类来指定文本的样式:

Class Transformation
is-capitalized 将单词首字母大写
is-lowercase 转化全部的单词为小写
is-uppercase 转化所有的单词为大写
{% include anchor.html name="文本粗细（Text weight）" %}

Class Weight
has-text-weight-light 文本weight调整为light
has-text-weight-normal 文本weight调整为normal
has-text-weight-semibold 文本weight调整为semi-bold
has-text-weight-bold 文本weight调整为bold
