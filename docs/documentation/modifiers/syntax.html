修改器语法（Modifiers syntax）

Bulma中许多元素都有可选择性的样式。要使用他们，你只需要把修改器类名增加到元素类中。
它们全部以is-或者has-开头。

我们从一个简单的button开始。使用"button" CSS class:

{{button_example}}

{% highlight html %}{{button_example}}{% endhighlight %}

通过添加"is-primary" CSS class,你可以修改颜色:

{{button_primary_example}}

{% highlight html %}{{button_primary_example}}{% endhighlight %}

你可以使用6种主要颜色的一种:

  • is-primary
  • is-link
  • is-info
  • is-success
  • is-warning
  • is-danger
