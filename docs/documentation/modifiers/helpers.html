辅助类（Helpers）

你可以使用 响应式辅助类来改变大多数的元素, 以此来在不同的浏览器宽度下获得对应的样式

Float is-clearfix 清除元素的子元素浮动
is-pulled-left 将元素移动到左侧
is-pulled-right 将元素移动到右侧
Spacing is-marginless 去除所有的外边距（margin）
is-paddingless 去除所有的内边距（padding）
Other is-overlay 覆盖第一个定位的父节点
is-clipped 添加overflow属性hidden
is-radiusless 去除所有的圆角（radius）
is-shadowless 去除所有的阴影（shadow）
is-unselectable 禁止元素被选中
is-invisible 添加显示属性（visible）hidden
