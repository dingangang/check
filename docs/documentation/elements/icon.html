图标（Icons）

Bulma兼容所有的字体图标库: Font Awesome, Material Design Icons, Open Iconic, Ionicons等。

{% include meta.html colors=true sizes=true variables=true %}

icon元素是一种对所有字体图标都适用的容器。因为图标通常需要一些时间来下载，所以你需要控制图标占据的空间，你可以使用icon元素作为可靠的正方形容器，它会防止页面在加载时“跳动”。

{% include snippet.html content=icon_example %}
此处的黄色背景只是作为演示，来标记图标容器区域。

默认的icon容器的尺寸会指定为 1.5rem x 1.5rem。图标本身的尺寸取决于你使用的图标库。例如Font Awesome图标库会继承字体尺寸。

{% include anchor.html name="Colors" %}

因为字体图标都是简单的文字，你可以使用颜色修饰器来改变图标颜色。

{% include snippet.html content=icon_color_example %} {% include anchor.html name="尺寸" %}
