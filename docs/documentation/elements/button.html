表单组和按钮列表的区别

按钮列表可以通过field is-grouped容器或者新的buttons容器来实现，它们还是有一些区别：

  • buttons有更简单的标记
  • buttons只能包含button元素
  • field is-grouped可以包含任意 的control输入框
  • field is-grouped会强制使所有的控件都在一行之中
  • 在field is-grouped中你可以展开其中的一个控件

总而言之，如果你只希望使用单纯按钮 组成的列表，那么就使用buttons。如果你需要使用更多的空间的元素。使用表单组.
